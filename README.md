# Foundation Todo App

JavaScriptの基礎を学ぶために、一から作成したシンプルなTodoアプリです。

## 概要

このプロジェクトは、JavaScriptがHTMLやCSSとどのように連携して動くのかを理解するための実践的な学習として作成しました。

現在のアプリでは、以下の機能を実装しています：

- タスクの追加
- タスクの削除
- 空の入力を防ぐ処理
- Enterキーでのタスク追加

## 目的

このプロジェクトの目的は、以下の基礎を理解することです：

- DOM操作
- イベント処理
- JavaScriptによる要素の生成と追加
- 入力値のバリデーション

## 動作の仕組み

### HTML
アプリの基本構造を定義しています：

- タイトル
- 入力欄
- 追加ボタン
- タスクリスト

### CSS
以下のデザインを担当しています：

- 全体レイアウト
- 中央配置されたカードUI
- 入力エリア
- ボタンのスタイル
- タスクリストの見た目

### JavaScript
以下の処理を実装しています：

- `getElementById()` による要素取得
- `addEventListener()` によるイベント処理
- `createElement()` による要素の生成
- `appendChild()` による要素の追加
- `remove()` による要素の削除
